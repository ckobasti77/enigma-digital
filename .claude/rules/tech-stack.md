# Tech Stack

- **Next.js 16** (App Router, React 19, Turbopack)
- **Tailwind CSS v4** with `@tailwindcss/postcss` (config in `tailwind.config.ts`, CSS variables in `app/globals.css`)
- **TypeScript** (strict mode, `@/*` path alias maps to repo root)
- **GSAP + @gsap/react** for scroll-driven animations
- **React Three Fiber + Drei** for 3D scenes (logo mark, service page floating objects)
- **simplex-noise** for the interactive wave background canvas
